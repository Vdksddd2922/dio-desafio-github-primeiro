# **comando para clonar repositório do git hub para pasta**

Comando: git clone

---

**comando para verificar se esta tudo certo com o repositório**
Comando: git status

**Comando para adicionar pasta "Atualizar"**
Comando: git add .

**Para adicionar um Comentário no repositório**
Comando: git commit -m "coloca a msg aqui"

**Comando para adicionar tudo pra nuvem do repositório**
Comando : git push origin main
